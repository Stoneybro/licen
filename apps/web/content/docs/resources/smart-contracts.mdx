## Verifying the Contract

You can read the full DataPolicy state from the 0G Explorer:

- **Registered datasets** — `getPolicy(datasetRoot)` returns the full Policy struct
- **Job state** — `getJob(jobId)` returns the current job state, escrow amount, and settlement info
- **Publisher earnings** — `claimableRoyalties(owner)` returns unclaimed balance

All settlement events are emitted on-chain and indexed by Envio. Every royalty payment, refund, and job completion is publicly auditable.
